第一道防線

📧 認明「董事長唯一信箱」標籤

資訊部已在郵件系統實施「視覺輔助辨識」，三秒鐘判斷真偽

✓

有標籤 = 正版

主旨顯示 [董事長唯一信箱] 才是真正來自董事長的信件

✗

無標籤 = 100% 詐騙

顯示「陳來助」、「來助」、「LJ」但沒有標籤，直接刪除，切勿回覆或點擊連結

⚠

紅色/黃色警示 = 危險

看到 External 或 垃圾郵件警示，立即提高警覺

第二道防線

📱 LINE 帳號「鐵壁防禦」七步驟

拿起手機，依照下列步驟強化設定
操作路徑：LINE 主頁 → 右上角齒輪 ⚙️ 設定

1

補齊帳號核心資料

設定 → 我的帳號

  • ✓ 確認「電話號碼」已綁定
  • ✓ 確認「電子郵件」與「密碼」已設定

⚠️ 密碼設定後無法查看，忘記只能重設

2

管理登入裝置

設定 → 我的帳號 → 登入中的裝置

檢查清單中是否有「非本人」裝置，若有請立刻按「登出」

📚 查看官方教學 →
3

關閉「允許自其他裝置登入」

🔥 最重要！

設定 → 我的帳號 → 允許自其他裝置登入

❌ 只用手機傳 LINE：務必關閉，100% 杜絕遠端盜用

✅ 需使用電腦版：保持開啟，但須定期檢查登入裝置

4

網頁登入雙重認證

設定 → 我的帳號 → 開啟「網頁登入雙重認證」

登入第三方網頁時，手機會跳出確認通知，防止帳密外洩後被盜用

5

啟用生物辨識

設定 → 我的帳號 → Face ID / Touch ID

用生物辨識解鎖，無需輸入密碼，避免密碼被鍵盤側錄

📚 查看官方教學 →
6

App 本機鎖

設定 → 隱私設定 → 開啟「密碼鎖定」

防止手機借人或遺失時，LINE 對話內容直接被看光

7

聊天加密 Letter Sealing

設定 → 隱私設定 → 開啟「Letter Sealing」

確保訊息點對點加密。換新手機時請依正規流程轉移

🆘 黃金救援

懷疑被盜帳號怎麼辦？

不慎掃描了不明 QR Code，或發現 LINE 突然無法登入、亂傳訊息？請立即行動！

📱

情況 A：手機還能打開 LINE

還有機會自救！

  1. 1

    踢除駭客

    設定 → 我的帳號 → 登入中的裝置 → 登出所有裝置

  2. 2

    更改密碼

    設定 → 我的帳號 → 密碼 → 設定新密碼

  3. 3

    關閉權限

    暫時關閉「允許自其他裝置登入」

🚨

情況 B：LINE 已被登出

駭客已改密碼，需官方介入

  1. 1

    立即回報

    填寫 LINE 官方問題反應表單，申請停權以減少親友受害

  2. 2

    官方救援

    參考下方連結中的「填寫客服表單」步驟
